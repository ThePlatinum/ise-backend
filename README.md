# ISE

Find the best talent for your projects!

## Setting up

01. `git clone ` the project and `cd` into project directory
02. `composer install`
03. `cp .env.example .env`
04. `php artisan key:generate`
05. `php artisan migrate --seed`
06. `php artisan serve`
